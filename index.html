Track expenses and understand your student budget.

This app helps a student add daily expenses, set a monthly budget, and read a simple financial summary in one place.

About

About This Project

Student Budget Tracker was created to support basic money management for students. It records expenses, converts supported currencies to RWF, and shows whether the spending is still within the monthly budget.

What The Website Can Do
